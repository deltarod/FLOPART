## Installing
This section will explain how to install pyFLOPART

### With Pip

`python3 -m pip install git+https://github.com/deltarod/FLOPART.git`

### From Sources

1. `git clone https://github.com/deltarod/FLOPART.git`
2. `cd FLOPART/`
3. `python3 -m pip install .`
